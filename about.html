Jurusan Informatika

Proyek ini melibatkan pembuatan sebuah website sederhana dengan tiga halaman: index.html, about.html, dan contact.html. Website ini memiliki menu navigasi yang menghubungkan ke setiap halaman. Halaman contact.html berisi form dengan input field untuk nama, email, kategori, pesan, dan checkbox untuk menerima balasan. Website ini menggunakan file CSS tunggal styles.css untuk mengatur gaya elemen HTML dan halaman. Proyek ini menunjukkan kemampuan dasar HTML dan CSS, termasuk membuat menu navigasi, mengatur gaya elemen HTML, dan menggunakan selektor CSS untuk menargetkan elemen dan halaman tertentu.
